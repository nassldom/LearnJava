<b>Part 7: Arrays (Kapitel 60-68)</b>

In diesem Teil lernen wir, wie man viele Daten des gleichen Typs in einer Liste (Array) speichert.

<b>1. Einführung in Arrays (Kapitel 60)</b>
Ein Array ist eine Sammlung von Werten mit einer <b>festen Größe</b>. Der Index beginnt immer bei <b>0</b>.

[CODE]
int[] zahlen = new int[5]; // Erzeugt ein Array für 5 ganze Zahlen
zahlen[0] = 10;            // Erster Wert
zahlen[4] = 50;            // Letzter Wert (bei Größe 5)
[/CODE]
- <b>zahlen.length:</b> Gibt die Anzahl der Elemente zurück (hier 5).

<b>2. Arrays durchlaufen (Kapitel 61)</b>
Um alle Werte eines Arrays zu verarbeiten, nutzen wir Schleifen.

[CODE]
// Klassische for-Schleife
for (int i = 0; i < zahlen.length; i++) {
    System.out.println(zahlen[i]);
}

// Enhanced for-Schleife (for-each)
for (int x : zahlen) {
    System.out.println(x);
}
[/CODE]

<b>3. Arrays von Objekten (Kapitel 65)</b>
Du kannst auch Arrays erstellen, die Objekte (z.B. Strings oder eigene Klassen) speichern.
<i>Achtung:</i> Die einzelnen Plätze im Array sind am Anfang <b>null</b>!

<b>4. Zweidimensionale Arrays (Kapitel 67)</b>
Man kann sich 2D-Arrays wie eine Tabelle mit Zeilen und Spalten vorstellen.

[CODE]
int[][] tabelle = new int[3][4]; // 3 Zeilen, 4 Spalten
tabelle[0][0] = 1;
[/CODE]

<b>5. StringBuffer (Kapitel 68)</b>
Ein <i>StringBuffer</i> ist wie ein veränderbarer String. Er ist effizienter, wenn man einen Text oft bearbeiten oder verlängern muss (mit <i>.append()</i>).
