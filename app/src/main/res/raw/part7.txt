<b>Part 7: Arrays (Kapitel 60-68)</b>

In diesem Teil lernst du, wie man viele Daten des gleichen Typs in einer Liste (Array) speichert und verarbeitet.

<b>1. Einführung in Arrays (Kapitel 60)</b>
Ein Array ist eine Sammlung von Werten mit einer festen Größe. Der Index beginnt immer bei 0.
[CODE]
int[] zahlen = new int[5]; // Array für 5 ganze Zahlen
zahlen[0] = 10;           // Erster Wert
zahlen[4] = 50;           // Letzter Wert (bei Größe 5)
[/CODE]
Die Länge erhältst du mit <code>zahlen.length</code>.

<b>2. Arrays durchlaufen & Algorithmen (Kapitel 61)</b>
Um alle Werte zu verarbeiten, nutzt man Schleifen.
[CODE]
for (int i = 0; i < zahlen.length; i++) {
    System.out.println(zahlen[i]);
}

// Alternativ: Enhanced for-Schleife (for-each)
for (int x : zahlen) {
    System.out.println(x);
}
[/CODE]

<b>3. Arrays als Parameter (Kapitel 63-64)</b>
Arrays können an Methoden übergeben werden. Da es sich um Objekte handelt, arbeitet die Methode auf dem Original-Array.
[CODE]
public void verdopple(int[] arr) {
    for (int i = 0; i < arr.length; i++) {
        arr[i] *= 2; // Original wird verändert!
    }
}
[/CODE]

<b>4. Arrays von Objekten (Kapitel 62 & 65)</b>
Du kannst auch Arrays erstellen, die Objekte (z.B. Strings oder eigene Klassen) speichern.
<b>Achtung:</b> Die einzelnen Plätze im Array sind am Anfang <code>null</code>!
[CODE]
String[] namen = new String[3];
namen[0] = "Alice";
[/CODE]

<b>5. Lineare Suche (Kapitel 66)</b>
Um ein Element in einem unsortierten Array zu finden, prüft man jedes Element nacheinander.
[CODE]
int gesucht = 50;
for (int i = 0; i < zahlen.length; i++) {
    if (zahlen[i] == gesucht) {
        System.out.println("Gefunden an Index: " + i);
    }
}
[/CODE]

<b>6. Zweidimensionale Arrays (Kapitel 67)</b>
Man kann sich 2D-Arrays wie eine Tabelle mit Zeilen und Spalten vorstellen.
[CODE]
int[][] tabelle = new int[3][4]; // 3 Zeilen, 4 Spalten
tabelle[0][0] = 1;
[/CODE]

<b>7. StringBuffer (Kapitel 68)</b>
Ein <code>StringBuffer</code> ist wie ein veränderbarer String. Er ist effizienter, wenn man einen Text oft bearbeiten oder verlängern muss.
[CODE]
StringBuffer sb = new StringBuffer("Hallo");
sb.append(" Welt");
System.out.println(sb.toString());
[/CODE]

<b>Merksatz:</b>
Arrays haben eine feste Größe. Musst du oft Elemente hinzufügen oder löschen, sind andere Strukturen (wie <code>ArrayList</code>) besser geeignet.
