<b>TEIL 7 – Arrays (Kap. 60–68)</b>

<b>Kapitel 60 – Einführung in Arrays</b>

Ein <b>Array</b> ist eine Liste von Werten desselben Typs mit <u>fester Größe</u>.
• Index beginnt bei <b>0</b>.
• Zugriff: <i>stuff[0]</i>.

[CODE]
int[] stuff = new int[5]; // Array für 5 Zahlen
stuff[0] = 23;
[/CODE]

<u>Wichtig:</u> <i>array.length</i> gibt die Größe zurück.


<b>Kapitel 61 – Array-Algorithmen</b>

Arrays durchlaufen mit Schleifen:

<b>Klassisch:</b>
[CODE]
for (int i=0; i < arr.length; i++) { ... }
[/CODE]

<b>Enhanced (for-each):</b>
[CODE]
for (int wert : arr) { ... }
[/CODE]


<b>Kapitel 62 & 63 – Arrays & Parameter</b>

Arrays können als Instanzvariablen in Klassen genutzt werden. Wenn du ein Array an eine Methode übergibst, wird die <b>Referenz</b> übergeben. Die Methode kann also die <u>Werte im Original-Array ändern</u>!


<b>Kapitel 64 & 65 – Methoden & Objekt-Arrays</b>

Du kannst auch Arrays von Objekten (z.B. Strings) erstellen:
<i>String[] namen = new String[5];</i>

<u>Achtung:</u> Die Zellen sind am Anfang <b>null</b>! Immer erst prüfen:
<i>if (namen[i] != null) { ... }</i>


<b>Kapitel 66 – Lineare Suche</b>

Prüfe jedes Element nacheinander, bis das gesuchte gefunden ist. Wenn nicht gefunden, gibt man oft <b>-1</b> zurück.


<b>Kapitel 67 – Zweidimensionale Arrays (2D)</b>

Wie eine Tabelle mit <u>Zeilen und Spalten</u>:
<i>int[][] tabelle = new int[3][5];</i> // 3 Zeilen, 5 Spalten
Zugriff: <i>tabelle[zeile][spalte]</i>


<b>Kapitel 68 – StringBuffer</b>

Da Strings unveränderlich sind, ist das Zusammenbauen vieler Texte in einer Schleife langsam. Nutze dafür <b>StringBuffer</b> (oder <i>StringBuilder</i>):

[CODE]
StringBuffer sb = new StringBuffer();
sb.append("Hallo");
sb.append(" Welt");
String ergebnis = sb.toString();
[/CODE]
