<b>TEIL 4 – Verzweigungen und Schleifen (Kapitel 16–25)</b>

<b>Kapitel 16 – Die if-Anweisung</b>

Mit der <b>if-Anweisung</b> triffst du Entscheidungen im Programm: Ein bestimmter Codeblock wird nur ausgeführt, wenn eine Bedingung wahr (true) ist.

<b>Zweiseitige Entscheidung (if-else):</b>
„Wenn Bedingung erfüllt, dann A, sonst B".

[CODE]
import java.util.Scanner;

public class BoxOffice {
    public static void main( String[] args ) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Wie alt bist du? ");
        int alter = scanner.nextInt();

        if ( alter >= 18 ) {
            System.out.println("Du darfst den Film sehen.");
        } else {
            System.out.println("Du bist noch nicht alt genug.");
        }
    }
}
[/CODE]

<b>Vergleichsoperatoren:</b>
• <b>==</b> gleich, <b>!=</b> ungleich
• <b><</b> kleiner, <b>></b> größer
• <b><=</b> kleiner-gleich, <b>>=</b> größer-gleich

<b>Kapitel 17 – Einseitige Verzweigung</b>

Manchmal soll nur im positiven Fall etwas passieren. Dafür braucht man kein else.

[CODE]
import java.util.Scanner;

public class CarPurchase {
    public static void main( String[] args ) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Jahreseinkommen: ");
        double einkommen = scanner.nextDouble();
        double preis = 25000.0;

        if ( einkommen > 50000 ) {
            preis = preis * 0.95; // 5% Rabatt
            System.out.println("Sie erhalten Rabatt!");
        }
        System.out.println("Kaufpreis: " + preis);
    }
}
[/CODE]

<b>Kapitel 18 – Boolesche Ausdrücke & Logische Operatoren</b>

Mit logischen Operatoren kombinierst du Bedingungen:
• <b>&& (UND)</b>: Beide müssen wahr sein
• <b>|| (ODER)</b>: Mindestens eine muss wahr sein
• <b>! (NICHT)</b>: Kehrt den Wert um

<b>Kurzschlussauswertung:</b> Java bricht ab, wenn das Ergebnis feststeht (z.B. bei &&, wenn links schon false ist).

<b>Kapitel 19 – Schleifen und die while-Anweisung</b>

Eine Schleife wiederholt Code, solange eine Bedingung wahr ist. Die <b>while-Schleife</b> prüft die Bedingung <u>vor</u> dem Durchlauf.

[CODE]
public class LoopExample {
    public static void main( String[] args ) {
        int zahl = 1;
        while ( zahl <= 5 ) {
            System.out.println("Zahl: " + zahl);
            zahl = zahl + 1; // Wichtig: Aktualisierung!
        }
    }
}
[/CODE]

<b>Kapitel 20 – Zählschleifen</b>

Zählschleifen laufen eine festgelegte Anzahl oft durch.

[CODE]
public class SumLoop {
    public static void main( String[] args ) {
        int summe = 0;
        int i = 1;
        while ( i <= 100 ) {
            summe = summe + i;
            i = i + 1;
        }
        System.out.println("Summe 1 bis 100: " + summe); // 5050
    }
}
[/CODE]

<b>Kapitel 21 – Verschachtelte Schleifen</b>

Eine Schleife innerhalb einer anderen Schleife. Die innere Schleife läuft bei jedem Durchlauf der äußeren komplett neu ab.

[CODE]
public class Pyramid {
    public static void main( String[] args ) {
        int i = 1;
        while ( i <= 5 ) {
            int j = 1;
            while ( j <= i ) {
                System.out.print("* ");
                j++;
            }
            System.out.println();
            i++;
        }
    }
}
[/CODE]

<b>Kapitel 22 – Schleifen mit Wächterwert (Sentinel)</b>

Wird verwendet, wenn die Anzahl der Eingaben unbekannt ist. Ein Sonderwert (z.B. -1) beendet die Schleife.

<b>Wichtig:</b> Priming Read (Eingabe vor der Schleife).

[CODE]
import java.util.Scanner;

public class SentinelLoop {
    public static void main( String[] args ) {
        Scanner scan = new Scanner(System.in);
        System.out.print("Zahl (-1 zum Ende): ");
        int n = scan.nextInt();

        while ( n != -1 ) {
            System.out.println("Quadrat: " + (n * n));
            System.out.print("Nächste Zahl: ");
            n = scan.nextInt();
        }
    }
}
[/CODE]

<b>Kapitel 23 – Ergebnisgesteuerte Schleifen</b>

Die Schleife läuft, bis ein berechnetes Ziel erreicht ist (z.B. Zinseszins).

[CODE]
public class Investment {
    public static void main( String[] args ) {
        double kapital = 1000.0;
        int jahre = 0;
        while ( kapital < 2000.0 ) {
            kapital *= 1.05; // 5% Zinsen
            jahre++;
        }
        System.out.println("Verdoppelt nach " + jahre + " Jahren.");
    }
}
[/CODE]

<b>Kapitel 24 – Die for-Schleife</b>

Die <b>for-Schleife</b> ist die kompakte Form der Zählschleife: <i>for(Initialisierung; Bedingung; Aktualisierung)</i>.

[CODE]
public class ForLoop {
    public static void main( String[] args ) {
        for ( int i = 1; i <= 5; i++ ) {
            System.out.println("i = " + i);
        }
    }
}
[/CODE]

<b>Kapitel 25 – Weitere Aspekte der for-Schleife</b>

• <b>Rückwärts zählen:</b> <i>for(int i=10; i>=1; i--)</i>
• <b>Schrittweite ändern:</b> <i>i += 2</i>
• <b>Verschachtelte for-Schleifen:</b>

[CODE]
public class Table {
    public static void main( String[] args ) {
        for ( int i = 1; i <= 3; i++ ) {
            for ( int j = 1; j <= 3; j++ ) {
                System.out.print(i * j + "	");
            }
            System.out.println();
        }
    }
}
[/CODE]

<b>Goldene Regel:</b>
• Anzahl bekannt? → <b>for</b>
• Anzahl unbekannt? → <b>while</b>
