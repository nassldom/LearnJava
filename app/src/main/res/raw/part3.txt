<b>PART 3 – Daten & Ein-/Ausgabe (Kap. 8–15)</b>

<b>Kapitel 8 – Primitive Datentypen</b>

Java kennt acht primitive Datentypen. Die vier wichtigsten für den Einstieg:

<b>int</b>: Ganze Zahlen (ca. -2,1 Mrd. bis +2,1 Mrd.)
<b>long</b>: Sehr große ganze Zahlen
<b>double</b>: Gleitkommazahlen (Dezimalzahlen)
<b>boolean</b>: Wahrheitswert (true oder false)
<b>char</b>: Einzelnes Zeichen (z.B. 'A')

Primitive Typen speichern ihren Wert <u>direkt im Speicher</u> – kein Objekt, keine Referenz. Das macht sie schnell und speichersparsam. In Java muss jede Variable einen <b>festen Typ</b> haben. Das nennt sich <u>statische Typisierung</u>.

<b>Codebeispiel (Variable.java):</b>

[CODE]
public class Variable {
    public static void main( String[] args ) {
        int alter = 21;
        double preis = 9.99;
        boolean istAktiv = true;
        char buchstabe = 'A';

        System.out.println("Alter: " + alter);
        System.out.println("Preis: " + preis);
    }
}
[/CODE]

<b>Typische Fehler:</b>
• <i>int zahl = 3.5;</i> → Compilerfehler, weil 3.5 ein double ist.
• <i>boolean flag = 1;</i> → In Java nicht erlaubt (nur true/false).


<b>Kapitel 9 – Variablen und Zuweisung</b>

Eine Variable ist ein <u>benannter Speicherplatz</u>.
• <b>Deklaration</b> = Typ + Name (z.B. <i>int punkte;</i>)
• <b>Initialisierung</b> = erster Wert (z.B. <i>punkte = 100;</i>)
• <b>Zuweisung</b> = Wert ändern (z.B. <i>punkte = 150;</i>)

Der <b>= Operator</b> ist keine mathematische Gleichung, sondern eine <b>Zuweisung</b>: Der Wert rechts wird in die Variable links gespeichert.


<b>Kapitel 10 – Arithmetische Operatoren</b>

<b>+</b> Addition, <b>-</b> Subtraktion, <b>*</b> Multiplikation, <b>/</b> Division, <b>%</b> Modulo (Rest).

<u>Achtung Ganzzahldivision:</u> Wenn beide Zahlen <i>int</i> sind, wird der Rest einfach abgeschnitten! <i>10 / 3</i> ergibt <b>3</b>, nicht 3.33.

<b>Operatorpriorität:</b> Punkt vor Strich (*, /, % vor +, -). Klammern () ändern die Reihenfolge.


<b>Kapitel 11 – Objektdaten (Strings)</b>

Neben primitiven Typen gibt es <b>Referenztypen</b> (Objekte). Der wichtigste ist <b>String</b>. Ein String speichert Text und bietet viele Methoden:

[CODE]
String name = "Java";
System.out.println(name.length()); // Länge
System.out.println(name.toUpperCase()); // Großbuchstaben
[/CODE]

<u>Wichtig:</u> Strings immer mit <b>.equals()</b> vergleichen, niemals mit ==!


<b>Kapitel 12 – Ein- und Ausgabe (Scanner)</b>

Für die Eingabe von der Tastatur nutzt man die Klasse <b>Scanner</b>:

[CODE]
import java.util.Scanner;
// ...
Scanner scanner = new Scanner(System.in);
int zahl = scanner.nextInt();
[/CODE]

<b>Methoden:</b> <i>nextInt()</i> für Zahlen, <i>nextLine()</i> für ganzen Text.


<b>Kapitel 13 – Mathematische Funktionen (Math)</b>

Java bietet die Klasse <b>Math</b> für Berechnungen:
• <b>Math.sqrt(x)</b>: Quadratwurzel
• <b>Math.pow(x, y)</b>: x hoch y
• <b>Math.PI</b>: Die Kreiszahl Pi

<u>Hinweis:</u> <i>Math.sin</i> und <i>Math.cos</i> erwarten das Argument im <b>Bogenmaß</b> (Radiant), nicht in Grad! Umrechnung mit <i>Math.toRadians()</i>.


<b>Kapitel 14 & 15 – Formatierte Ausgabe</b>

Für schöne Ausgaben (z.B. 2 Nachkommastellen):

<b>1. DecimalFormat:</b>
<i>DecimalFormat df = new DecimalFormat("0.00");</i>
<i>System.out.println(df.format(12.345)); // 12,35</i>

<b>2. printf():</b>
<i>System.out.printf("Preis: %.2f Euro", 19.99);</i>

Platzhalter: <b>%d</b> (int), <b>%f</b> (double), <b>%s</b> (String), <b>%n</b> (Zeilenumbruch).
