PART 13 – JavaFX Grafik
(Kap. 120–121)

Grundstruktur:
[CODE]
public class MeinProgramm extends Application {
    public void start(Stage primaryStage) {
        Circle kreis = new Circle(100, 100, 50);
        kreis.setFill(Color.BLUE);

        Group root  = new Group(kreis);
        Scene scene = new Scene(root, 400, 300);
        primaryStage.setScene(scene);
        primaryStage.show();
    }
    public static void main(String[] args) {
        launch(args);
    }
}
[/CODE]

Transformationen:
[CODE]
shape.setRotate(45);      // 45° drehen
shape.setTranslateX(100); // nach rechts verschieben
shape.setScaleX(2.0);     // horizontal skalieren
[/CODE]

Koordinatensystem:
• Ursprung (0,0) ist oben links
• X wächst nach rechts
• Y wächst nach unten

Layout-Panes:
• Group    – freie Positionierung per Koordinaten
• StackPane – Elemente übereinander stapeln
• HBox     – Elemente nebeneinander
• VBox     – Elemente untereinander
